# Modern Java Features Test Cases

This directory contains test cases for modern Java features.

## Test Files

1. **OptionalTest.java** - Tests Optional with `map()` and `orElse()`
2. **CompletableFutureTest.java** - Tests CompletableFuture with `thenApply()`
3. **GenericExtends.java** - Tests generic interfaces with `extends`
4. **MultipleInterfaces.java** - Tests classes implementing multiple interfaces

## Expected Behavior

The tool should:
- Handle Optional operations and stub Optional methods
- Handle CompletableFuture operations
- Properly stub generic interfaces with extends relationships
- Handle multiple interface implementations
